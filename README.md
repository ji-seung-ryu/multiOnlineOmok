## Getting Started


### Prerequisites

* git
  ```sh
  sudo yum install -y git
  ```

* java
  ```sh
  sudo amazon-linux-extras install java-openjdk11
  ```

  
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ji-seung-ryu/multiOnlineOmok.git
   ```
2. move to multiOnlineOmok
   ```sh
   cd multiOnlineOmok
   ```
3. modify authority
   ```sh
   sudo chmod 777 ./gradlew
   ```
4. build 
   ```sh
   ./gradlew build
   ```
